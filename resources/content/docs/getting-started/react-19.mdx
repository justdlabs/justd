## Recharts
Rechart v2.15.0 now fully support **React 19**, but you still need to add the following override to your `package.json`. Read more about it [here](https://github.com/recharts/recharts/releases/tag/v2.15.0).

```
"overrides": {
  "react-is": "^19.0.0"
}
```
