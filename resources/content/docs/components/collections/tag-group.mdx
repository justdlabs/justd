## Basic
A tag group is a focusable collection of labels, categories, keywords, filters, or other elements, equipped with keyboard navigation, selection, and removal functionalities.

<How toUse="collections/tag-group-with-remove-demo" />

## Intents
The default intent is `primary`. You can also use `secondary`, `success`, `warning`, and `danger`.
<How toUse="collections/tag-group-intent-demo" />

## Controlled
Manage the values of the tag group through the `value` prop.
<How toUse="collections/tag-group-controlled-demo" />

## Href
Incorporate a hyperlink within the tag group using the `href` prop.
<How toUse="collections/tag-group-href-demo" />



## Shape
The tag group can be shaped to be a square or a circle. Just like the badge component, see [badge](/d/badge?#shape) documentation for more information.
<How toUse="collections/tag-group-shape-demo" />

## Disabled
Disable tags either by setting the `isDisabled` prop or using `disabledKeys` on the `TagGroup`.
